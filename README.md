# Image Encryption for Secure Image Transfer

## Project Overview
This project focuses on securing image transmission using a hybrid encryption approach combining AES (Advanced Encryption Standard) and RSA algorithms. It ensures secure and efficient image transfer over networks.

## Features
- Image encryption using AES algorithm
- Secure key exchange using RSA
- Image compression for efficient transmission
- Secure sharing of encrypted images

## Technologies Used
- Python
- Cryptography Library
- OpenCV / Pillow
- AES & RSA Algorithms
